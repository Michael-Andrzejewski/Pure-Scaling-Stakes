# Pure Scaling Stakes

A Balatro mod that adds 4 new stakes with only blind scaling increases and no debuffs. All stakes play like White Stake — no eternal, rental, or perishable jokers — just higher score requirements.

## Stakes

| Stake | Scaling Level | Equivalent To |
|-------|--------------|---------------|
| White-Blue | 2 | Red Stake's blind scaling |
| White-Red | 3 | Green Stake's blind scaling |
| White-Black | 4 | Cryptid Pink Stake's blind scaling |
| White-Void | 5 | Cryptid Verdant Stake's blind scaling |

## Requirements

- [Balatro](https://store.steampowered.com/app/2379780/Balatro/)
- [Lovely](https://github.com/ethangreen-dev/lovely-injector) (>= 0.8) — the mod injector that makes Balatro modding possible
- [Steamodded](https://github.com/Steamodded/smods) (>= 1.0.0-BETA) — the modding framework/API

Cryptid is **not** required, but this mod is fully compatible with it.

## Installation

1. Install Lovely and Steamodded if you haven't already.
2. Copy the `ScalingStakes` folder into your Balatro mods directory:
   ```
   %AppData%/Balatro/Mods/
   ```
3. Launch Balatro. The 4 new stakes will appear in the stake selection screen.
